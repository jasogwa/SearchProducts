# Installation 

## Required!

* Node : Recommended node version: 16+
* Django


## BackEnd

*   ``cd aiioproject``
*   ``python manage.py migrate``
*   ``python manage.py seed``
*   ``python manage.py runserver``

## FrontEnd

*   ``cd FrontEnd``
*   ``npm install``
*   ``npm run dev``

# Run

*   ``Open : http://localhost:5173/  on your browser``
